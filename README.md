## About

This is a Full Stack Application made with [Laravel Framework](https://laravel.com/docs), and [React.js](https://react.dev/). It features a CRUD for Materials and its Categories (Many to One) with the following characteristics:

- SOLID
- Strategy Pattern
- Form requests
- Services

Laravel is accessible, powerful, and provides tools required for large, robust applications.
